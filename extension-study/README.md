# 推广研究 / Extension Study

对母因子的六个预注册推广方向,全部在诚实口径(时点严格、含退市、真实成交、一次性样本外、25 次试验族级校正)下证伪;唯一证实的命题——尾盘存在与早盘正交的信息窗口——不可货币化。完整协议、证据与证伪工具箱见报告。

Six preregistered extensions of the base factor, all falsified under the honest
protocol (point-in-time, survivorship-free, execution-honest, one-shot OOS,
family-wide correction over 25 trials). The one confirmed proposition — an orthogonal
tail-of-day information window exists — does not monetize. Full protocol, evidence,
and the falsification toolkit are in the reports.

| # | 方向 / Direction | 结果 / Verdict |
|---|---|---|
| A1 | 早盘离散度 / early-session dispersion | 证伪:波动率代理 / volatility proxy |
| A2 | 额加权广度 / amount-weighted breadth | 证伪:level 污染 / level contamination |
| A3 | 窗口内路径 / intra-window path shape | 证伪:样本外变号 / OOS sign flip |
| B1 | 尾盘广度 / tail-of-day breadth | 命题证实,不可货币化 / confirmed, not monetizable |
| B3 | 长假首日 / post-holiday reopening | 证伪:机理梯度反向、与母策略共线 / gradient reversed, collinear |
| B5 | 波动率择时 / volatility timing | 证伪:统计显著但无决策价值 / significant, no decision value |
